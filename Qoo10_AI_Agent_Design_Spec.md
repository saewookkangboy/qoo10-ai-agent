## 1. 디자인 원칙

### 1.1 핵심 원칙

### 4.3 카드 컴포넌트 설계

#### 기본 카드 구조
```
┌─────────────────────────────────┐
│  [아이콘]  카드 제목              │
├─────────────────────────────────┤
│                                 │
│      카드 내용                   │
│                                 │
│  ┌─────────────────────────┐   │
│  │  세부 정보 또는 액션     │   │
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

#### 카드 타입별 상세 설계

##### 1. 종합 점수 카드
- **크기**: 2열 (Desktop), 1열 (Mobile)
- **내용**:
  - 큰 숫자로 점수 표시 (예: 75점)
  - 점수 색상: 80+ (Success), 60-79 (Warning), <60 (Error)
  - 점수 설명: "양호" / "개선 필요" / "긴급 개선"
  - 간단한 트렌드 표시 (선택)

##### 2. 우선순위 카드
- **크기**: 2열 (Desktop), 1열 (Mobile)
- **내용**:
  - 우선순위 배지: High / Medium / Low
  - 개선 항목 개수 표시
  - 빠른 액션 버튼

##### 3. 상품 상세 분석 카드
- **크기**: 전체 너비
- **내용**:
  - 각 분석 항목별 점수 (별점 + 숫자)
  - 항목별 간단한 설명
  - "자세히 보기" 링크
  - 시각적 진행 바 또는 레이더 차트

##### 4. 매출 강화 아이디어 카드
- **크기**: 전체 너비
- **내용**:
  - 우선순위별 그룹핑 (High / Medium / Low)
  - 각 아이디어는 체크박스 형태
  - 아이디어 클릭 시 상세 설명 모달
  - "액션 아이템으로 저장" 기능

##### 5. 체크리스트 카드
- **크기**: 전체 너비
- **내용**:
  - 카테고리별 체크리스트
  - 진행률 표시 (예: 60% 완료)
  - 완료/미완료 상태 시각화
  - 체크박스 인터랙션

##### 6. 경쟁사 비교 카드
- **크기**: 전체 너비
- **내용**:
  - Top 10 상품 비교 테이블
  - 가격, 리뷰, 이미지 수 등 비교
  - 차별화 포인트 하이라이트

## 5. 인터랙션 디자인

### 5.1 마이크로 인터랙션

#### 버튼 호버
- **효과**: 배경색 변경 (0.2초), 약간의 그림자 증가
- **커서**: pointer

#### 카드 호버
- **효과**: 그림자 증가 (Level 1 → Level 2), 약간의 상승 (translateY: -2px)
- **트랜지션**: 0.2s ease-out

#### 로딩 애니메이션
- **스피너**: 회전하는 원형 인디케이터
- **스켈레톤 로더**: 카드 형태의 펄스 애니메이션

#### 성공/에러 피드백
- **토스트 메시지**: 화면 상단 또는 하단에 슬라이드 인
- **지속 시간**: 3초 (자동 닫힘)
- **애니메이션**: 0.3s ease-out

### 5.2 페이지 전환
- **전환 효과**: 페이드 + 슬라이드 (0.3s)
- **방향**: 좌→우 (진행), 우→좌 (뒤로가기)

### 5.3 스크롤 동작
- **부드러운 스크롤**: 활성화
- **스크롤 인디케이터**: 긴 페이지의 경우 진행률 표시

## 6. 정보 아키텍처

### 6.1 네비게이션 구조
```
홈 (URL 입력)
  └─ 분석 결과
      ├─ 종합 점수
      ├─ 상품 상세 분석
      ├─ Shop 분석 (Shop URL인 경우)
      ├─ 매출 강화 아이디어
      ├─ 체크리스트
      └─ 경쟁사 비교
```

### 6.2 정보 우선순위
1. **1순위**: 종합 점수 및 우선순위 (즉시 확인)
2. **2순위**: 매출 강화 아이디어 (액션 가능)
3. **3순위**: 상세 분석 결과 (이해 및 검증)
4. **4순위**: 체크리스트 및 경쟁사 비교 (참고)

## 7. 접근성 (Accessibility)

### 7.1 WCAG 2.1 AA 준수
- **색상 대비**: 텍스트와 배경 최소 4.5:1
- **키보드 네비게이션**: 모든 인터랙티브 요소 접근 가능

### 7.2 키보드 단축키

- **Esc**: 모달 닫기
- **Tab**: 다음 요소로 이동
- **Shift + Tab**: 이전 요소로 이동
